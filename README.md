# Kittygram 

Приложение, в котором пользователи могут добавлять своих котов с фото и описанием, взаимодействовать с ними через веб-интерфейс и API. Реализована авторизация, контейнеризация, CI/CD и автоматические тесты.

## Описание проекта

**Kittygram** — это мини-сервис про домашних животных. Зарегистрированные пользователи могут добавлять записи о своих котах, указывать характеристики и фото, просматривать записи других. Только автор записи может её редактировать или удалять. Между фронтендом и бэкендом настроено API, все сервисы упакованы в Docker-контейнеры, и настроены рабочие процессы CI/CD.

## Основные возможности

- Регистрация и вход пользователей.
- Добавление, редактирование и удаление информации о котах (только для авторов).
- Просмотр списка котов других пользователей.
- Связь между фронтендом и бэкендом через REST API.
- Контейнеризация с помощью Docker и Docker Compose.
- Автоматическое тестирование через pytest.
- Настроенный CI/CD (GitHub Actions). 

## Польза проекта

Этот проект полезен как демонстрация полного цикла разработки веб-приложения:

- показывает работу с Django REST API и клиентской частью;
- иллюстрирует аутентификацию и авторизацию пользователей;
- учит правильно упаковывать сервисы в контейнеры;
- содержит пример настройки CI/CD и автоматических тестов;
- служит готовой базой для расширения функциональности под собственные задачи.

## Как установить на локальный компьютер

**1. Клонировать репозиторий**

```
git clone https://github.com/polinenysh/kittygram_final.git
cd kittygram_final
```

**2. Подготовить окружение**

Убедитесь, что установлены:

•	Docker
  
•	Docker Compose

**3. Настроить переменные окружения**

Скопируйте пример:
```
cp .env.example .env
```
и заполните поля в .env по своему окружению.

**4. Запустить в Docker**
```
docker compose up --build
```
Docker соберёт образы и запустит все сервисы проекта.

**5. Открыть в браузере**

После запуска приложение будет доступно по адресу: 127.0.0.1:9000

**Тесты можно запускать локально внутри контейнера или в виртуальном окружении:**

pytest
